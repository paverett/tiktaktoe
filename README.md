# TikTakToe

TikTakToe is an online two person Tic Tac Toe Or Noughts and Crosses game.

## Requirements

Node

## Installation

Use npm to install the required packages.

```
npm install
```

## Usage

```
node index.js
```
Once running open localhost:/3000 in two tabs to see it in action!
